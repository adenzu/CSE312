PART C

THIS PART DOES DEMAND SPECIFIC USER INPUTS

EACH PROGRAM ONLY NEEDS AND ACCEPTS ONLY POSITIVE INTEGERS
FOLLOW THE INSTRUCTIONS EXACTLY FOR INPUTS FOR RESPECTIVE PROGRAMS

PRINTED NUMBERS ARE *ALWAYS* IN HEXADECIMAL FORMAT 
ENTERED NUMBERS ARE *ALWAYS* IN DECIMAL FORMAT

THERE IS NO TIME LIMIT FOR INPUTS TO BE ENTERED 


FOLLOW STEPS
    OPEN TERMINAL IN THIS DIRECTORY 
    RUN 'make install' TO CREATE THE .iso FILE
    CREATE YOUR OS PROFILE IN YOUR FAVOURITE VIRTUAL MACHINE WITH THIS .iso PATH
    RUN 'make run'
    WAIT FOR OS TO START
    ENTER VALID INPUTS WHENEVER IT IS PROMPTED ON SCREEN
    YOU CAN EXIT WHEN ALL TASKS ARE FINISHED
